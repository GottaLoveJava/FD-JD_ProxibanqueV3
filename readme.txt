	************************************
*****  	    Description de l'application:   	*****
	************************************

ProxibanqueSI est une application bancaire.
Ce système permet au conseiller de clientèle d'afficher la liste
des clients de l'agence, de consulter les comptes de chaque client
et de réaliser des virements compte à compte.

	************************************
*****  	          Documentation:	        *****
	************************************

Pour avoir accès à la documentation veuillez suivre les indications suivantes :

1) Ouvrez le dossier 'javadoc'.

2) Double cliquez sur 'index.html'.

	************************************
***** 	     Execution de l'application:    	*****
	************************************

Veuillez suivre les indications suivantes:

1) Placer le fichier 'FD-JD_ProxibanqueV3.war' dans le dossier de déploiement de votre serveur TomCAT.

2) Démarrez votre base de données MySQL sur votre serveur.

3) Démarrez votre serveur TomCAT.

4) Editez le fichier tomcat-users.xml afin de créer vos identifiants et password

5) A l'aide de votre navigateur, tapez dans le champ URL : http://localhost:8080/FD-JD_ProxibanqueV3/
